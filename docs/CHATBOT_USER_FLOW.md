## Overview

Chatbot Bengkelin adalah asisten virtual berbasis BotMan yang membantu pengguna dalam:
- Melakukan booking bengkel
- Mencari dan membeli spare part
- Mengecek status pesanan
- Menemukan bengkel terdekat
- Melihat riwayat booking
- Memberikan rating/ulasan
- Menjawab FAQ

### Teknologi yang Digunakan
- **BotMan** v2.8 - Framework chatbot untuk Laravel
- **BotMan Web Driver** v1.5 - Web interface driver
- **Laravel Sanctum** - Authentication
- **Laravel Cache** - Context/session management

## Intent Recognition

Chatbot menggunakan multi-layer intent matching:

### 1. Payload-Based (Highest Priority)
Direct match dari payload button clicks.

### 2. Pattern Matching
Regex patterns untuk format spesifik:
- `status TRANS-xxx`
- `rate {id} {stars} "comment"`
- `koordinat -6.xxx,106.xxx`

### 3. Keyword Matching
Pencocokan kata kunci dengan synonym mapping:

| Intent | Keywords (ID) | Keywords (EN) |
|--------|---------------|---------------|
| booking | pesan, reservasi, jadwal, servis | book, reserve |
| product | produk, spare part, barang, item | product, part |
| search | cari, temukan, lihat | find, search |
| cancel | batal, batalkan, keluar | cancel, stop |
| help | bantuan, tolong, cara | help, how |
| nearby | dekat, terdekat, sekitar | near, nearby |

### 4. Fuzzy Matching (Lowest Priority)
Menggunakan Levenshtein distance dengan threshold 70%:
- `boking` â†’ booking
- `bokking` â†’ booking
- `produk` â†’ product
